羊了个羊 无脑爽版 过关不是目的，目的是摧残方块( iOS )

Posted on 2022-09-23 In hack

先看图

Pre

整个过程，不安装三方app，不需要提供uid，按着说明来就行。
注意：整个wifi的设置务必按照文中的顺序和时机，并且关闭任何vpn，否则可能会失败。

使用步骤：

分两个步骤：

    1. 安装证书（让苹果允许修改游戏数据数据）
    1. 修改数据
1. 安装证书
  • 1.1 注意关闭手机任何vpn!（重要）
    连接到一个wifi网络，进入该wifi的设置，
    点击最下面的http代理 -> 选中间的手动 -> 填入相关信息
    服务器 216.238.109.242
    端口 8889（因为随时会变，可以问我）
    点右上角保存
  • 1.2 点击这里 下载证书
    会提示你下载一个证书文件，点击确定下载
  • 1.3 来到手机设置 -> 通用 -> VPN和设备管理
    点进去会有一个下载好的profile文件，点击会弹出安装提示，右上角点击安装，输入锁屏密码，再次点击弹出的安装，之后点击右上角完成，安装完成。
  • 1.4 来到手机设置 -> 通用 -> 关于 -> 证书信任设置
    点击进去，有一个mitmproxy，点击右侧开关至绿色，弹出来根证书提示，点击继续
  • 1.5 至此证书安装完成。
  • 1.6 断开服务器
    回到wifi设置，http代理 -> 选择关闭 -> 点击右上角保存
2. 修改数据

2.1 先从小程序删除羊了个羊，（不是右上角的小圆点关闭退出，是将小程序删除）
2.2 再次进入游戏，进入到所有省的排名界面，先“不要”点下面的“进入羊圈”（重要）

2.3 再次连接到服务器，(按上面1.1操作)
2.4 回到羊了个羊，点击“进入羊圈”，稍等片刻一直到进入游戏界面（有云飘过，开始消块）。注意这里进游戏时间可能会比平时慢，15s以下都是正常的，务必耐心，超过时间了还没进去就再点一下，重复操作，直到进入游戏画面。
2.5 一旦进入游戏界面后断开服务器，（按上面1.6操作）

2.6 回到游戏，继续游玩，来到第二关会发现变得超级简单。

注意：整个wifi的设置务必按照上面的顺序和时机，并且关闭任何vpn，否则可能会失败。
